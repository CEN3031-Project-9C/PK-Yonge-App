About the Training and Practice Tests

The purpose of the training and practice tests is for students to become familiar with the CBT system, functionality, and item types; the training/practice tests are not intended to guide classroom instruction. Descriptions and response instructions for each item type are included in the Training and Practice Tests Quick Guide. Users should refer to the tutorials on each item or the guide to familiarize themselves with the different features and response instructions for each item type.

Training tests were released in 2014 and contain test items for multiple grade levels (subjects for EOC). Grade-level and subject-specific practice tests are being developed for release beginning in Fall 2015. Please see the list for the availability of practice tests. Please note that the training tests will be available until all practice tests are published.

Other information

Answer keys for English Language Arts (ELA) Reading and Mathematics training and practice tests that list the item, correct answer, and grade level/subject of each item are also available for each published training/practice test. ELA Writing rubrics are available under the FSA Resources section

Users must use the Secure Browser or a supported Web browser to access the Training and Practice Test site. The supported Web browsers for the training and practice tests are Firefox 10–36, Chrome 41–44, Safari 5–8, or Internet Explorer 10–11.

Test Features

How to use the features provided by this test-taking site.

This online practice test provides a number of different tools while taking the test in order to provide students with everything they will be able to use on an actual EOC exam.

_

Navigation

On the top left of the test sceen, you will find a number of different buttons for navigating through the exam:
-The "Back" and "Next" buttons allow you to move forwards and backwards among questions.
-The "Pause" button allows you to temporarily pause the test and return to it later.
-The "End Test" button will permanantly end the current test you are taking.

Tools

On the top right of the test sceen, you will find a number of different tools to use on the test:
-The "Formula" button brings up an approved formula sheet you can view while taking the exam.
-The "Calculator" button will bring up a basic calculator which can be used during the exam.
-The "Line Reader" button will highlight each line of the question in order.
The "Zoom In"and "Zoom Out" buttons will zoom in and zoom out of the page when pressed.

Test Features

How to use the features provided by this test-taking site.

This online practice test provides a number of different tools while taking the test in order to provide students with everything they will be able to use on an actual EOC exam.

_

Navigation

On the top of the test sceen, you will find a number of different buttons for navigating through the exam:
-The back and next buttons allow you to move forwards and backwards among questions.
-The dropdown menu allows you to move to any question of your choosing.
-The "Submit Test" button will permanantly end the current test you are taking.

Tools

On the top left of the test sceen, you will find a number of different tools to use on the test:
-The "Calculator" button will bring up a basic calculator which can be used during the exam.
-The "Formula Sheet" button brings up an approved formula sheet you can view while taking the exam.
-The "Notepad" button brings up a notepad that will be saved with this question, in case you'd like to come back to the question and review your notes.
-The "Mark for Review" marks a question for review, which shows up while on the question, and in the dropdown question menu.

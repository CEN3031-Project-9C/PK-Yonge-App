Question Types

How to answer the different types of questions you might see on an exam.

There are five main types of questions you may be asked on the algebra EOC exam:

-Multiple choice questions provide a number of possible answers, and you must select the correct one.

-Fill in the blank questions require you to solve a problem on your own, and put the correct answer(s) in the text box(es).

-Multiple select questions provide a number of possibles answers, and you must select which ones are correct.

Accessing the Test

Where and how to take the practice EOC exam.

In order to access the test, please go to the main page of this website.

_

Once there, the test can be accessed either by pressing the large "Begin Practice Test" button on the homepage, or by selecting "Choose Test" on the top menu.

It is important to note that the test is best and most easily taken using any modern web browser on a computer. It will likely not be functional on a phone or tablet device.
